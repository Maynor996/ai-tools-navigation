# AI 工具导航站 - GitHub Pages 自动化项目

一个基于 GitHub Pages 和 Jekyll 的 AI 工具导航网站，支持每日自动更新内容。

## 🚀 特性

- ✅ **自动更新**：每天自动检查AI平台状态并更新页面
- ✅ **真实链接**：集成真实可用的AI工具平台
- ✅ **状态监控**：实时检测平台可用性和响应时间
- ✅ **GitHub Pages**：完全免费托管，自定义域名支持
- ✅ **响应式设计**：移动端友好的现代化界面
- ✅ **Jekyll 静态生成**：快速加载，SEO 友好
- ✅ **API支持**：包含AI API中转服务
